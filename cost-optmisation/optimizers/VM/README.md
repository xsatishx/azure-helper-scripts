# Azure VM Cost Optimizer
**Author:** [Satish Balakrishnan](mailto:satishbal@microsoft.com)  
## Script

`01_azure_vm_optimizer_v1.sh`

This optimizer analyzes Azure virtual machines using the baseline cost report generated by the core cost collector. It enriches VM cost data with VM metadata, CPU metrics, pricing lookups, and rightsizing recommendations.

---

## Prerequisites

Before running this script, run the core collector first and generate:

```text
FinalRankedReport.csv
```

The VM optimizer requires `FinalRankedReport.csv` as its main input.

Example core collector flow:

```bash
cd ../../core
chmod +x 00_azure_cost_data_collector_v1.sh
./00_azure_cost_data_collector_v1.sh subs.txt FinalRankedReport.csv
```

The script also requires:

- Azure CLI
- Python 3
- Azure login completed
- Reader access to target subscriptions
- Permission to read Azure Monitor VM metrics

---

## How to Run

From the `optimizers/vm/` folder:

```bash
chmod +x 01_azure_vm_optimizer_v1.sh
./01_azure_vm_optimizer_v1.sh ../../core/FinalRankedReport.csv VM_Optimization_Output.csv
```

Optional example for top results:

```bash
TOP_N=5 ./01_azure_vm_optimizer_v1.sh ../../core/FinalRankedReport.csv VM_Optimization_Output_top5.csv
```

---

## Output

```text
VM_Optimization_Output.csv
```

The output includes VM cost, SKU, region, CPU usage, pricing lookup status, estimated savings, and recommendation confidence.

---

## Notes

This script is read-only. It does not resize, stop, start, delete, or modify any virtual machines.
