# Azure Helper Scripts

A collection of helper scripts and assessment utilities for working with Microsoft Azure environments.

This repository contains scripts for Azure cost optimisation, resiliency assessment, and network/vWAN assessment activities. Each folder has its own README with more detailed usage instructions, prerequisites, and examples.

## Repository Structure

| Folder                 | Description                                                                             |
| ---------------------- | --------------------------------------------------------------------------------------- |
| `cost-optimisation`     | Scripts and guidance for analysing Azure cost optimisation opportunities.               |
| `ResiliencyAssessment` | Scripts and guidance for assessing Azure workload and platform resiliency.              |
| `vWAN-assessment`      | Scripts and guidance for reviewing Azure Virtual WAN and related network configuration. |

## How to Use

Start by opening the folder relevant to your task and follow the README inside that folder.

For example:

* For cost analysis, go to `cost-optimisation`
* For resiliency reviews, go to `ResiliencyAssessment`
* For Azure Virtual WAN reviews, go to `vWAN-assessment`

## Notes

These scripts are intended as helper tools and should be reviewed before running in any customer or production Azure environment. Always validate permissions, scope, and output before using the results for decision-making.

## Disclaimer

This repository is provided as-is for learning, automation, and assessment support. Test carefully before using in production environments.
